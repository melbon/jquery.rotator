# jquery.rotator
Testimonials Rotator
## Install
```javascript
<script src="jquery.min.js"></script>
<script src="../jquery.rotator.min.js"></script>
```
or 
```javascript
bower install rotator
```
## Usage
```javascript
$(":input").rotator();
```
## Configuration
#### delay
time next element wait before stating
```javascript
default: 5000
options: number
```
#### fadeSpeed
fade transition duration
```javascript
default: "slow"
options: "slow" / "fast" or number
```
